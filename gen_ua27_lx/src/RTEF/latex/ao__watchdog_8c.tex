\doxysubsection{Detailed Description}
Implements the Watchdog Active Object. 

This file defines the Watchdog Active Object, responsible for monitoring system activity through heartbeat messages and detecting unresponsive components.

The watchdog periodically checks system health and can take corrective action if a component fails to send a heartbeat within the defined timeout.

The implementation supports both Windows (via system timers) and Free\+RTOS (via task scheduling).
