\doxysubsection{Detailed Description}
Defines the Watchdog Active Object. 

This file provides the structure and function declarations for the watchdog active object ({\ttfamily \doxylink{structwatchdog__obj__t}{watchdog\+\_\+obj\+\_\+t}}). It integrates with the active object framework and utilizes a timer to monitor system health.

The watchdog ensures periodic checks are performed and can trigger recovery actions if a system component becomes unresponsive.
