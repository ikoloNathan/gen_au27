This file provides the structure and function declarations for the UDP active object ({\ttfamily udp\+\_\+obj\+\_\+t}). It integrates with the Active Object framework and facilitates event-\/driven communication over a UDP socket.

The UDP active object is only available for Windows-\/based systems ({\ttfamily \+\_\+\+WIN32}).
