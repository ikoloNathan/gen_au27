Detailed Description

Implements the Active Object pattern for message-driven execution.

This file provides functions for managing Active Objects, including initialization, message queue handling, dispatching, logging, and integration with different operating systems (Windows & FreeRTOS).

void unregister_active_object ( base_obj_t * me)

Unregisters an ActiveObject from the system registry.

This function removes an ActiveObject from the global registry when it is stopped. It ensures that the watchdog system no longer tracks the object, preventing false alerts about inactive objects.

Parameters

Note
