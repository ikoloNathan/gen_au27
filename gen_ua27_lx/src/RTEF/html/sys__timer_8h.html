Detailed Description

Provides an abstraction for system timers.

This file defines an interface for managing system timers, allowing registration of callback functions, enabling/disabling timers, and handling priority-based scheduling of timer callbacks.

It supports both Windows (using threads and waitable timers) and FreeRTOS (using software timers).
