Detailed Description

Defines the system active object and its state machine.

This file provides the structure and function declarations for the system active object (system_obj_t). It integrates with the active object framework and manages system states using a finite state machine (FSM).

The system active object is responsible for transitioning between different operational states and handling timed events.
